wine dataset 為一 classification（分類）問題，
在一個經典的 classfication 問題中，一筆資料（又稱一 data instance）將包含以下部分：

  • feature （或叫 attribute ）部分
    • 此筆資料的各項屬性
  • target（或叫 label）部分
    • 此筆資料的分類

Visualization¶

對 alcohol, malic_acid, ash 三個 feature，各自繪製一 scatter 圖 - 20%¶

  • title 為其 feature 名字
  • X 軸為其 target，並需對應 class_0 - class_2
  • Y 軸為該 feature 在對應的 target 下的分佈
  • 點需依照 target 上色，三個 target 需顏色不同

Hints:

  • pyplot.title
  • pyplot.scatter
  • pyplot.xticks
In [97]:
t_names = wine['target_names'] #class_0 - class_2 (xticks)
colors = ['purple', 'green', 'yellow']
x = wine['target']
y = np.split(wine['data'], 13, axis=1)

for i in range(3):
    plt.figure(figsize=(20, 10)) # 讓你的圖看起來大大的
    plt.title(wine['feature_names'][i])
    plt.xticks(np.arange(3), t_names)
    for j in range(0, 178):
        if x[j] == 0:
            plt.scatter(x[j], y[i][j], c='purple')
        elif x[j] == 1:
            plt.scatter(x[j], y[i][j], c='green')
        else:
            plt.scatter(x[j], y[i][j], c='yellow')
    plt.show() # 畫圖囉
